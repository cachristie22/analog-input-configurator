AQUAVX Pro Dictionary Reference

Site Dictionary
Name            (1): site name (31 chars)
DST             (2): daylight savings offset (0-2 hrs)
Timezone        (3): 0-26
MODBUS PollMode (7): 0=single register poll, 1=multi-register poll
MODBUS PollRate (8): poll rate (0-3600 secs, 0=500 msec)
MODBUS Slave ID (9): MODBUS Slave ID (1-247/255)
GetTimeMethod   (11): 0=Internet, 1=CELL, 2=GPS, 3=BLE, 4=None
MaxDataAge      (13): the maximum interval before runtime data is considered static (1-32000 secs)
LED Brightness  (51): set LED brightness (0-100%)
Display Timeout (53): turn off display after this time of inactivity (0-1440 mins)
MODBUS Master Dictionary
Name             (1): representative register name (31 chars)
Enable           (2): 0=disabled, 1=enabled
Link Type        (3): physical link type, 0=RTU, 1=TCP
SlaveID          (4): address of slave device to read from (1-247/255)
FunctionCode     (6): Read/Write function codes
RegisterAddress  (7): address of register in slave to read (1-65535)
RegisterType     (8): 0=reg16, 1=reg32, 2=bit16, 3=bit32, 4=packed16, 5=packed32, 6=float32
BitWeight        (9): scale factor (float)
Offset           (10): scale offset (float)
SignedRegister   (11): 0=unsigned, 1=signed
DisplayPrecision (12): decimal places to show in value (0-8)
ByteOrder(Endian)(13): 0=LSB first (little), 1=MSB first (big)
BitNumber        (14): for extracting single bit (0-32, 0=leave packed)
PackedMask       (15): mask unwanted bits in packed register (0x0-0xFFFFFFFF)
Options Dictionary
Block Alarm Dialog  (1): Enable to block all alarm dialog popups
LEDsOff             (3): Enable to turn off all LEDs except the GRN System LED
ShortVID            (17): Shorten AssetID to last 6 chars of IMEI
PersistentCELL      (28): Suspend CELL session instead of tearing down after transfer
MQTTProtocol        (29): Set telemetry to use MQTT protocol
Disable Auto Backup (30): Disable automatic backing up of config before updates
CELL Dictionary
AssetID              (1): Asset ID string (31 chars)
AcknowledgeType      (2): 0=none, 1=auto, 2=on-demand
PINGRate             (4): Send PING to server (0-1440 mins, 0=disable)
HostServerProtocol   (6): 0=UDP, 1=TCP, 2=SSL
SIM APN              (12): Access Point Name (63 chars)
SIM Username         (13): Username (63 chars)
SIM Password         (14): Password (63 chars)
MaxDailySMS          (18): Max daily SMS transmissions (0-25)
SMS911Phone#         (19): Phone number to send SMS 911 messages (15 chars)
FTP Dictionary
ReportType         (1): Get/Put FTP Report Type
PutPath            (2): FTP server write path (63 chars)
GetPath            (3): FTP server read path (63 chars)
GetFileName        (4): File name on FTP server (63 chars)
FTPServerIPAddress (7): FTP server IP address (127 chars)
FTPServerUserName  (8): FTP login username (63 chars)
FTPServerPassword  (9): FTP login password (63 chars)
Mode               (10): 0=active, 1=passive
SerialPort Dictionary
Name             (1): Representative port name
Enable           (2): 0=disabled, 1=enabled
PortMode         (3): 0=none, 1=debug, 2=cell, 3=master, 4=slave, 5=satellite, 6=bluetooth
RS232/RS485      (4): 0=RS232, 1=RS485
Baud             (5): Baud rate options: 1200, 2400, 4800, 9600, 19200, 38400, 57600, 115200
FlowControl      (15): 0=no flow control, 1=rts/cts flow control
Report Dictionary
EndOfDayOffset        (1): Offset from midnight for end of day report (0-1440 mins)
EndOfDayRate          (2): End of day report rate (0-1440 mins)
StdReportRate         (3): Standard report rate (0-1440 mins)
PushMaintInterval     (20): Max interval between successive PUSH messages (1-1440 mins)
CHAN Dictionary
Name              (1): Representative Channel name (31 chars)
Type              (3): 1=digital, 2=analog
Enable            (2): 1=enabled, 0=disabled
Reports           (4): 1=enabled, 0=disabled
SourceDataReg     (5): Source data register for this channel
DisplayPrecision  (20): Decimal positions to show in value (0-8)
Ethernet Dictionary
Host Name              (1): Representative Host Name (15 chars)
Default IP Address     (2): Default unit IP (127 chars)
Subnet Mask            (3): Subnet mask (xxx.xxx.xxx.xxx)
Gateway IP             (4): Gateway IP address (127 chars)
DHCP Client Enable     (5): 0=disable, 1=enable
Modbus TCP Server Enable (20): 0=disable, 1=enable
Modbus TCP Client Enable (30): 0=disable, 1=enable
ADC Dictionary
Name            (1): Representative ADC input name (31 chars)
InputType      (2): 0=none, 1=0-20mA, 2=4-20mA, 3=0-5v, 4=1-5v, 5=+/-600ACV, 6=+/-5AAC
DampingFactor  (3): Smooth noisy signals (0.0-1.0, float)
ZeroScaleEngr  (4): Zero scale engineering units (float)
FullScaleEngr  (5): Full scale engineering units (float)
DispPrecision  (6): Decimal places to show in values (0-8)
Pump Control Dictionary
Enable               (1): 0=disabled, 1=enabled
Mode                 (2): 0=drain, 1=fill
Alternation          (4): 0=automatic, 1=manual
Number Pumps         (3): 1=simplex, 2=duplex
Sensor Type          (7): 0=floats, 1=sensor
Debounce             (30): Debounce for all float inputs & setpoints (secs)
Start Delay          (31): Delay pump start (secs)
Alarm Function Block Dictionary
Name          (1): Representative alarm name (31 chars)
Enable        (2): 1 enables, 0 disables
Popup         (3): 1 enables alarm popups, 0 disables
Type          (4): 0=standard, 1=latched
SourceChan    (5): Value checked for alarm condition
DispPrecision (6): Decimal places for analog values (0-8)
AlarmDelay    (7): Time in alarm state before activation (0-999999 secs)
CallRoster    (8): 0=none, 1=standard, 2=priority
Digital Output Function Block Dictionary
Name                 (1): Representative digital output name (31 chars)
Enable               (2): 1 enables, 0 disables
SourceReg            (3): Data register determining output state
SrcActiveState       (5): Source active state (0-1)
OutputDataReg        (4): Output state written to this data register
OutputActiveState    (7): 0=deactivate, 1=activate
OutputDuration       (8): Max duration in active state (secs)
Analog Output Function Block Dictionary
Name                (1): Representative analog output name (31 chars)
Enable              (2): 1 enables, 0 disables
SourceReg           (3): Determines output value
DestinationReg      (4): Output value written here
AnalogZeroScale     (6): Scaling zero value (float)
AnalogFullScale     (7): Scaling full value (float)
DispPrecision       (5): Decimal places in values (0-8)
Timed Event Function Block Dictionary
Name            (1): Representative timed event name (31 chars)
Enable          (2): 1 enables, 0 disables
AutoStartOffset (3): Offset from midnight to start (mins)
RepeatInterval  (5): Time between repeats (mins)
RunDuration     (6): Run duration of timed event (secs)
Averages Function Block Dictionary
Name              (1): Representative name for this function block (31 chars)
Enable            (2): 1 enables, 0 disables
SourceReg         (3): Data register for computed average
SamplingInterval  (5): Time interval for samples (1-86400 secs)
AveragingInterval (6): Interval for computed average (1-1440 mins)
LowThreshold      (7): Discard samples below this threshold (float)
HighThreshold     (8): Discard samples above this threshold (float)
